# Swiss Changelog

## [Initial Release] - {PR_MERGE_DATE}

- **Swiss Transform** — type or paste a value (auto-filled from the selection / clipboard) and see every
  relevant swiss conversion at once: Base64, hex, Base58, hashes, `json → yaml/xml/Go struct`, JWT decode,
  HTML/URL escaping, slug & case conversion, IP math, colors, cron and time. Irrelevant operations are
  hidden automatically. Copy or paste any result.
- **Swiss Generate** — generate UUIDs (v4 / v7), passwords and lorem ipsum locally, with one-key regenerate.
- Detects the `swiss` CLI automatically (Homebrew / PATH) with a configurable path preference, and guides
  installation when it is missing.
